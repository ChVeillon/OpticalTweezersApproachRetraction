# OpticalTweezersApproachRetraction

This python code was written to process data from approach-retraction experiments using optical tweezers. 
One particle is fixed while the other is moved and the aim of the experiment is to measure the force as a function of distance between particles.
The force is measured on the fixed particle.
The code is designed for an experiment containing 10 cycles of approach/break/retractation/break.

For more detailed information regarding the physical principles and the methodology of this data processing, please refer my PhD thesis (written in French).
